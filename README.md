# Interactive Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript. It was created as part of a web development assignment to demonstrate frontend skills and basic interactivity using the DOM.

## Features

- Semantic HTML5 layout
- Responsive design using CSS Flexbox
- Projects showcase section
- Contact form with submission alert
- Dynamic hobby list (add and delete items)
- Live date and time displayed in the footer

## Technologies Used

- HTML5
- CSS3 (Flexbox, Box Model, Media Queries)
- JavaScript (DOM manipulation and event handling)

## Project Structure

## Sections Included

- Home: Hobby description with interactive benefits list  
- About: Short bio  
- Projects: Sample projects with descriptions  
- Contact: Form with submission alert  
- Footer: Social links and live date/time  

## How to Run

1. Download or clone the repository  
2. Open the project folder  
3. Open `index.html` in a web browser  

## Learning Outcomes

This project helped me understand how to structure webpages using HTML, style them with CSS, and add interactivity using JavaScript. I also learned how to manipulate the DOM and handle user events.
